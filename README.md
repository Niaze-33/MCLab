# 📱 Flutter UI Design Assignment

This project is a Flutter-based UI design that replicates a given layout using core Flutter widgets like **Container, Row, Column, and Alignment**.

---

## 🎯 Objective

The goal of this assignment is to build a visually structured UI layout by replacing given texts with custom data:

* Roll Number
* Registration Number
* District
* Blood Group
* Nickname

---

## 🧩 Features

* 📌 Custom top header with date
* 🎨 Multi-colored horizontal layout using `Row` and `Expanded`
* 🧾 Registration card with rounded corners
* 🔘 Floating circular button with gradient (Nickname)
* 📐 Proper spacing and alignment using `Column`

---

## 🛠️ Technologies Used

* Flutter
* Dart

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/YOUR_USERNAME/flutter-ui-design-assignment.git
   ```

2. Navigate to the project folder:

   ```bash
   cd flutter-ui-design-assignment
   ```

3. Run the app:

   ```bash
   flutter pub get
   flutter run
   ```
